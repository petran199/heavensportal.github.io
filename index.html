Project sex(male - female) recogition

In this application i obtained data based on user preferences on facebook. I obtained 16 features (for example favorite music genre, job etc.) from 100 men and 100 women. Then i used Genetic algorythm(for 4 different classifiers , knn,SVM,NN,RBF) to choose the best of them. Afterwards,i used data normalization for RBF and NN classifiers to avoid convergence problems. Finaly, given a random user preferences test set i checked out which of these 4 classifiers has the best recognition score.

Project "Coin Recognition"

On this project there are 3 different methods that a user can recognise the coin.The first approach uses 'lbp' (local binary pattern) method to recognise the coin showing the result on the screen. The second one uses 'ORB' (Oriented FAST and Rotated BRIEF) method,and the last one uses 'SIFT' (Scale-Invariant Feature Transform) method.
